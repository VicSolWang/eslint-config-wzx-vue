The custom vue eslint rules based on airbnb

# Installation

    npm install --save-dev eslint-config-wzx-vue

# Usage

Add eslint-config-wzx-vue to the "extends" in your .eslintrc.* file.

    {
        "extends": "eslint-config-wzx-vue"
    }
